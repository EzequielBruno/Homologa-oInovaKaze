**Descrição Resumida** (máx. 250 palavras):
Sistema web para gerenciamento completo do ciclo de vida de demandas de desenvolvimento de software em ambientes corporativos. O sistema permite que usuários solicitantes criem demandas, que passam por múltiplos níveis de aprovação (gerencial, comitê técnico e TI). Inclui funcionalidades de estimativa de horas e custos, faseamento de projetos, controle de backlog, visualização em Kanban, gestão de aprovações, histórico de ações, relatórios analíticos, agendamento de plannings e retrospectivas. O sistema implementa controle de permissões baseado em papéis (RBAC), autenticação de usuários, notificações em tempo real e dashboards executivos com métricas de performance.

## Passo a Passo para Registro no INPI

### 1. Preparação dos Documentos (1-2 semanas)
- Reunir toda documentação técnica (já fornecida nesta pasta)
- Fazer capturas de tela de todas as funcionalidades principais
- Exportar código fonte completo
- Organizar em arquivo ZIP ou PDF

### 2. Cadastro no Sistema do INPI (1 dia)
- Acessar: https://www.gov.br/inpi/pt-br
- Criar conta no sistema e-Marcas/e-Patentes
- Solicitar certificado digital (se for PJ) ou usar login gov.br (PF)

### 3. Emissão da GRU (1 hora)
- Gerar Guia de Recolhimento da União no site do INPI
- Código de serviço: 1810 (Registro de Programa de Computador)
- Valor: R$ 185,00 (pessoa física) ou R$ 370,00 (pessoa jurídica)
- Pagar em qualquer banco ou via internet banking

### 4. Preenchimento do Formulário (2-3 horas)
- Baixar formulário no site do INPI
- Preencher com os dados acima
- Assinar digitalmente ou reconhecer firma

### 5. Protocolo do Pedido (1 hora)
- Acessar sistema e-Software do INPI
- Fazer upload de todos os documentos
- Anexar comprovante de pagamento da GRU
- Protocolar pedido

### 6. Acompanhamento (30-90 dias)
- Aguardar análise do INPI
- Acompanhar pelo sistema e-Software
- Responder eventuais exigências

### 7. Certificado de Registro
- Após aprovação, baixar certificado digital
- Validade: 50 anos a partir da data de criação

## Observações Importantes

1. **Prazo de Proteção**: 50 anos a partir de 1º de janeiro do ano seguinte à criação
2. **Renovação**: Não é necessária renovação
3. **Transferência**: Pode ser transferido mediante averbação no INPI
4. **Sigilo**: O código fonte fica sob sigilo no INPI
5. **Prova de Autoria**: O registro serve como prova de autoria e anterioridade
6. **Patente vs Registro**: Software não pode ser patenteado, apenas registrado
7. **Proteção Internacional**: Para proteção internacional, registrar em cada país
